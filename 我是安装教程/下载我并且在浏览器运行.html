node.js的本质相当于一个类似与Tomcat的东东，也就是一个http服务器，原因：node把Chrome的V8 JS引擎封装起来了

1.安装node.js：

在百度上面直接node就可以了第一个就是node.js的中文官网进去就按下图的箭头顺序点击即可。安装傻瓜式安装点完即可

查看截图

然后我们只需要打开cmd命令行，运行 node -v 测试安装 即可

查看截图

2.环境的配置：

node的默认环境为国外服务器，看到国外服务器就能感觉的到运行的速度很超级的慢，所以我们提供了一个国内服务器，下面配置国内淘宝的镜像服务

在cmd中 npm install -g cnpm --registry=https://registry.npm.taobao.org 即可（下载资源用的）

下面所有的npm都可以用cnpm 自行替换即可 也可以不用按照个人的喜好

查看截图

3.安装VUE：

安装过cnpm之后就可以用cnpm了，也可以用npm 但是npm的速度超级慢。

我们使用cnpm info vue 或者 npm info vue 就可以查看我们可以安装的vue的版本了

查看截图

我们使用 cnpm install vue来安装vue ，注意 cnpm能干的事npm都可以。 我们也可以使用 cnpm install vue#xxx xxx为版本号来安装不同的版本

查看截图

4.安装VUE-CLI（创建vue项目用的工具）：

我们使用xue有两种方法一种是直接使用script src的方式，另外一种就是这次讲的第一种不讲

vue的项目相对来说挺复杂的，如果你使用的是HbuildrX的话可以直接创建，省略该步骤

安装vue-cli 很简单和上述安装方式一样npm uninstall vue-cli -g 即可

查看截图

截图没有丢 截图在最上面，由于我安装过我再次安装就提示这了~

5.创建项目：

创建项目我们使用的是vue的cli，使用HbuliderX的请忽略，这里只是简单的创建，更高深的可以看官网，也可以度娘

创建也是沙雕式的创建，vue create hello-world（需要等很长时间，可能是我的电脑不好） 就会创建一个叫hello-world的vue项目了

查看截图

这是里面的目录结构：

  • node_modules

    ↑：这里都是node下载的资源主要有vue的代码，如果在开发的时候需要用，bootstrap的话安装后也会下载到这里面，但是一般情况下不会吧这个玩意上传到git上，因为费流量

  • public

  • src

  • .gitignore

  • babel.config.js

  • package.json

  • package-lock.json

  • README.md
